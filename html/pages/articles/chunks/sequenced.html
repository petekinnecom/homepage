Sequenced

Story

Sequenced is an Android app written as I worked through the book Hello, Android. The app is a simple number puzzle game. Although the project was written in one afternoon, I was able to team up with a graphic designer to help with the artwork. He provided me mockups and .png's; all of the code is mine. We are currently working together on Transient 2.

Highlights

  • Written in Java using the Android SDK.
  • Worked with Java's font decoration system to dynamically create the number tiles.
  • Pretty! Yay!

Ideas for Future
