Reversi

Story

Written for a Combinatorial Game Theory class I took in graduate school, this program was built to analyze a modified-rules version of Reversi. While it does include some game tree analysis, most of the strategies are based on heuristic models. Note: this program was not built for the public; it was built to help my partner and I analyse Reversi strategies.

Highlights

  • Built using Java and SQLite.
  • Simple UI
  • Console for game and analysis information
  • User can play as either color against a strategy of his/her choice.
  • Able to build SQL databases of all game states (given enough time...)

Ideas for Future

  • Multi-thread the game tree analysis
  • Make game board auto-resize to fit window.
  • Introduce better game logic control and user messages.
