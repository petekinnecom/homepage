Net_Rain

Story

My first foray into realtime Java and netcode, Net_rain is inspired by one of my favorite arcade games "Super Puzzle Fighter 2 Turbo". Intended to be a tetris-like button masher, large quantities of falling bricks fill up your game lane while you try to remember which button combination does what. The multiplayer aspect works flawlessly on a LAN, but struggles over the internet, mostly due to issues with firewalls and routers. Net_Rain's rebirth as a single-player game for Android is in the works.

Highlights

  • Server/Client model hosts up to 4 connections at a time.
  • Uses both TCP and UDP for data transmission.
  • Realtime game loop using Model, View, Controller structure.
  • Features a chat lobby and game window.
  • Basic collision detection between falling bricks.
