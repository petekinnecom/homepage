Transient

Story

My first major project for Android, Transient is a sidescrolling, platform jumping game with a twist. Written as a way for me to familiarize myself Android, Transient quickly broadened my scope of abilities following a workthrough of the book Core Java 1 & 2. My experience is being put to work in my production of Transient 2.

Highlights

  • Written in Java using the Android SDK.
  • Level data stored in SQLite database.
  • Realtime game loop uses a multi-threaded Model, View, Controller structure.
  • Optimized for performance on older phones.
  • Reasonable physics and collision detection.
  • Simple, yet clear game flow and control.

Ideas for Future

  • Have a graphic designer update the artwork.
  • Scale down the difficulty of the levels.
